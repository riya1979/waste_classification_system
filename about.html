About This Project

This is an AI-powered Waste Classification Web Application developed using Deep Learning and Flask. The system classifies waste images into different categories such as Organic and Recyclable using a Convolutional Neural Network (CNN).

Problem Statement

Manual waste segregation is time-consuming and inefficient. This project aims to automate the classification process using Artificial Intelligence to improve environmental sustainability.

Model Architecture

  • Architecture: Custom CNN
  • Input Size: 224 x 224
  • Optimizer: Adam
  • Loss Function: Binary cross entropy
  • Activation Function: ReLU & Sigmoid
  • Epochs: 10

⚙ Technologies Used

  • Python
  • Flask
  • TensorFlow & Keras
  • HTML, CSS

Features

  • Image Upload
  • Real-time Image Preview
  • Confidence Score Visualization
  • Interactive Web Interface
